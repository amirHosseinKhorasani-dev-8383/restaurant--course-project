# Restaurant Website (Course Project)

This project was built as part of a training course to practice front-end development using Sass.

## Technologies
- HTML
- CSS
- Sass (SCSS)

## Description
A static restaurant website created during a training course.
All code was written by me while following the instructor, with a focus on structured styling, reusable components, and clean UI design using Sass.

## Features
- Modular and maintainable styles with Sass
- Multi-section landing page
- Clean and modern restaurant UI
- Responsive-friendly layout
